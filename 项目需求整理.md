## 1. 项目概述

### 1.1 项目背景

锁杆机器人装配用于集装箱锁杆的自动化装配，用于替代人工装配，提高自动化水平。当前的需求仅是满足锁杆对位装配的工序环节。

本项目是一个基于 **海康机器人（80kg型号）** 和 **梅卡曼德Mech-Eye Ultra Nano 350视觉系统** 的集装箱锁杆自动化对位装配系统。系统通过RTDE+TCP协议控制机器人，结合视觉系统获取箱门装配孔位坐标，实现锁杆与箱门的自动对位和螺栓穿装。

**项目定位**：从最小单元 Demo 展示阶段，进入功能完善和实验室验证阶段。

### 1.2 项目目标

| 目标 | 描述 |
|------|------|
| **自动化装配** | 替代人工锁杆装配，提高装配效率和一致性 |
| **精准对位** | 实现锁杆与箱门 12 组孔位的整体精准对位 |
| **工艺集成** | 完整实现锁杆装配的全流程自动化 |
| **工业级可靠性** | 具备断线重连、异常处理、安全联锁等工业级特性 |
| **多品牌兼容** | 支持海康、UR等多品牌机器人适配 |

### 1.3 项目范围

**包含范围**：
- 机器人控制系统（软件）
- 视觉系统集成（通信与坐标转换）
- 锁杆工装定位逻辑
- 快换模块控制逻辑
- 工艺流程编排

**不包含范围**：
- 锁杆工装机械设计与制造
- 快换模块硬件设计与制造
- 3D 视觉系统硬件（外购）
- 集装箱箱门生产线改造
- 螺母安装与拧紧工序
- 工业现场粉尘防护、保养等（实验室阶段）

## 2. 应用场景与工艺说明

### 2.1 应用场景

| 场景 | 描述 |
|------|------|
| **集装箱制造车间** | 箱门装配工位，替代人工完成锁杆装配 |
| **自动化产线** | 集成到箱门装配生产线，实现无人化作业 |
| **多型号兼容** | 支持不同型号集装箱的锁杆装配 |

### 2.2 现有锁杆工艺

**锁杆结构**：
- 长度：约 2m
- 螺栓数量：12 颗
- 排布方式：4 颗正方形均布 → 2 颗等距 → 2 颗等距 → 4 颗正方形均布
- 不同型号差异：中间段螺栓排布可能不同（2 颗或 4 颗）

**人工装配流程**：
1. 操作人员人工对齐锁杆与箱门安装位置
2. 将 12 颗螺栓依次穿入匹配孔位
3. 所有螺栓穿装到位后，工作人员在箱门反面安装螺母并拧紧固定

### 2.3 新设计系统组成

| 单元 | 功能 | 说明 |
|------|------|------|
| **锁杆工装** | 预装定位锁杆与配套螺栓 | 机械臂搭载，一次性完成锁杆与箱门 12 组孔位的整体对位，精度±0.1mm，ISO A50接口 |
| **锁杆适配机械臂** | 运动执行 | 海康80kg型号机器人（具体型号后续提供），通信协议RTDE+TCP |
| **快换模块** | 机械臂与工装快速拆装对接 | ISO 9409-1-A50标准，气动方式，到位/关锁/解锁三种DI状态（高电平有效） |
| **视觉检测模块** | 识别箱门装配孔位并输出坐标 | 梅卡曼德Mech-Eye Ultra Nano 350，安装在法兰盘上（眼在手），通过mech-vision软件识别 |

## 3. 核心技术指标

| **快换时间** | ≤ 30s | 工装拆装时间 |
| **连续运行时长** | ≥ 8h | 无故障连续运行 |

### 3.3 安全指标

## 7. 通信协议规范

### 7.1 机器人通信

| 项目 | 说明 |
|------|------|
| 协议 | RTDE（Real-Time Data Exchange）+ TCP自定义协议 |
| 通信方式 | TCP/IP |
| RTDE端口 | **待确认（海康）**（UR5为30002） |
| TCP端口 | **待确认（海康）**（后续提供） |
| 数据格式 | **待确认（海康）**（UR5为RTDE二进制 + TCP文本） |
| 刷新率 | **待确认（海康）**（UR5为125Hz） |
| 单位 | **待确认（海康）**（UR5为位置米、旋转弧度、速度m/s） |

**关键依赖**：海康机器人通信协议文档为P0级阻塞项，必须在阶段二启动前提供，否则无法进行适配器层详细设计

### 7.2 视觉系统通信

| 项目 | 说明 |
|------|------|
| 协议 | TCP Client（触发）+ JSON文件（坐标输出） |
| 编码方式 | ASCII |
| 远程地址 | 127.0.0.1:50000 |
| 命令格式 | `p,1,0,j1,j2,j3,j4,j5,j6,x,y,z,rx,ry,rz\r` |
| 终止符 | `\r` |
| 响应格式 | 状态码（1-7） |
| 坐标输出 | JSON文件 |

**状态码定义**：

| 状态码 | 含义 |
|--------|------|
| 1 | 成功 |
| 2 | 非法指令 |
| 3 | 工程未加载 |
| 4 | 无点云 |
| 5 | 无结果 |
| 6 | 规划失败 |
| 7 | 其他错误 |

### 7.3 坐标转换协议

**手眼标定方式**：眼在手（Eye-in-Hand），相机安装在法兰盘上

**手眼标定外参**：
- 格式：平移向量 + 四元数（x, y, z, qw, qx, qy, qz）
- 单位：位置（米），四元数（无量纲）
- 状态：**待重新标定**（原外参在UR5上标定，换用海康机器人后相机安装位置和法兰盘不同，需重新标定）

**工具偏移**：
- 定义：锁杆工装定位基准点相对于法兰盘中心的坐标
- 标定方式：手动测量，精度±0.1mm
- 状态：待标定

## 8. 核心工艺流程

### 8.1 锁杆自动对位装配主流程

```
┌─────────────────────────────────────────────────────────────────────┐
│  1. 人工完成锁杆、螺栓预装，整体放置于物料定位工位 A                    │
├─────────────────────────────────────────────────────────────────────┤
│  2. 机械臂搭载 3D 视觉模块移动至拍摄点位，采集箱门孔位图像                 │
│     完成法兰坐标系与工具坐标系姿态校准（姿态标定工序 B）                   │
├─────────────────────────────────────────────────────────────────────┤
│  3. 姿态校准完成后再次拍摄成像，提取 12 个装配孔位坐标数据                 │
│     解算得到装配基准中心坐标 C                                          │
├─────────────────────────────────────────────────────────────────────┤
│  4. 机械臂运动至物料工位 A，通过快换模块抓取锁杆工装                      │
│     携带工装运动至基准中心坐标 C                                         │
├─────────────────────────────────────────────────────────────────────┤
│  5. 以中心坐标 C 为基准沿 Z 轴直线进给                                   │
│     将锁杆螺栓一次性穿入箱门对应孔位（穿装距离由视觉计算）                    │
│     通过位置反馈判断穿装是否到位                                         │
├─────────────────────────────────────────────────────────────────────┤
│  6. 设备输出装配到位信号至控制系统，待机等候下一动作指令                    │
│     （视觉确认穿装作为后续优化功能）                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### 8.2 快换模块控制流程

```
抓取流程：
    机械臂到达物料工位 → 发送夹紧指令（DO-01=1）→ 等待夹紧信号（DI-03=1）→ 确认抓取成功

松开流程：
    装配完成 → 发送松开指令（DO-02=1）→ 等待松开信号（DI-04=1）→ 确认工装释放
```

### 8.3 系统启动初始化流程

```
系统启动 → 自动初始化 → 状态检查（机器人连接、视觉系统、快换模块）
         → 坐标系校准（外参检查+偏移检查）→ 进入待机状态
```

## 9. 坐标转换详细设计

### 9.1 转换流程

```
相机坐标系（视觉输出）
    ↓ 格式：[x, y, z, qw, qx, qy, qz]（四元数）
    ↓ 应用手眼标定外参 T_CF（平移向量 + 四元数）
法兰坐标系下位姿
    ↓ 应用工具偏移 T_TF（手动测量的工装基准点偏移）
工具坐标系下位姿（目标位姿）
    ↓ 计算与当前TCP位姿的偏差 delta_xyz, delta_rpy
    ↓ 工具坐标系相对移动指令
机器人执行移动
```

### 9.2 转换公式

**步骤1：相机坐标到法兰坐标转换**

```
T_CF = 手眼标定外参（4×4齐次变换矩阵）
法兰位姿 = T_CF × 相机位姿
```

**步骤2：法兰坐标到工具坐标转换**

```
T_TF = 工具偏移矩阵（4×4齐次变换矩阵）
工具位姿 = T_TF × 法兰位姿
```

**步骤3：计算相对偏差**

```
当前TCP位姿：P_current
目标位姿：P_target
delta = P_target - P_current（在工具坐标系下）
```

### 9.3 数据格式

**视觉输出格式（JSON文件）**：
```json
{
    "holes": [
        {"x": 0.1, "y": 0.2, "z": 0.05, "qw": 1.0, "qx": 0.0, "qy": 0.0, "qz": 0.0},
        {"x": 0.12, "y": 0.22, "z": 0.05, "qw": 1.0, "qx": 0.0, "qy": 0.0, "qz": 0.0}
    ],
    "center": {"x": 0.11, "y": 0.21, "z": 0.05, "qw": 1.0, "qx": 0.0, "qy": 0.0, "qz": 0.0},
    "detection_time": 2.5
}
```

**手眼标定外参格式**：
```python
# 平移向量（米）+ 四元数（w, x, y, z）
T_CF = [0.118718, 0.040764, 0.085525, 0.707883, -0.010377, -0.009907, 0.706184]
```

**工具偏移格式**：
```python
# 平移向量（米）+ 旋转（弧度）
```

### 10.1 安全联锁

| 联锁项 | 触发条件 | 响应动作 |
|--------|----------|----------|

## 13. 异常与故障处理

### 13.1 异常分类

| 类别 | 异常项 | 处理策略 |
|------|--------|----------|
| **通信异常** | 机器人连接断开 | 自动重连（最大 3 次） |
| | 视觉系统无响应 | 超时重试（最大 3 次） |
| | 网络中断 | 等待网络恢复，重试连接 |
| **运动异常** | 移动失败 | 记录错误码，进入报警状态 |
| | 位置超出范围 | 禁止执行，返回错误 |
| | 速度超限 | 自动限制到安全范围 |
| **视觉异常** | 无点云（状态码 4） | 重试拍摄 |
| | 无结果（状态码 5） | 检查工件位置，重试 |
| | 规划失败（状态码 6） | 记录日志，人工干预 |
| **工艺异常** | 螺栓穿装异常 | 停止进给，报警提示 |
| | 工装夹紧失败 | 重试夹紧，失败则报警 |
| | 锁杆定位偏移 | 重新标定，调整坐标 |

### 13.2 故障恢复流程

```
报警状态 → 故障诊断（日志分析）→ 故障排除（人工或自动）→ 清除错误 → 复位到待机 → 重新启动
```

### 13.3 错误码定义

| 错误码 | 含义 | 处理方式 |
|--------|------|----------|
| E001 | 机器人连接失败 | 检查网络，重试连接 |
| E002 | 视觉系统连接失败 | 检查视觉系统状态 |
| E003 | 移动指令执行失败 | 检查目标位置，重试 |
| E004 | 坐标转换失败 | 检查外参配置 |
| E005 | 工装夹紧失败 | 检查快换模块 |
| E006 | 螺栓穿装异常 | 检查位置反馈，人工检查 |
| E007 | 安全门打开 | 关闭安全门，重新启动 |
| E008 | 急停触发 | 复位急停按钮，重新启动 |
